Follow these steps to get your running map live on the internet!

## Step 1: Create a GitHub Account (if you don't have one)
1. Go to https://github.com
2. Click "Sign up"
3. Follow the registration process

## Step 2: Create a New Repository
1. Log in to GitHub
2. Click the "+" icon in the top right
3. Select "New repository"
4. Repository name: `my-running-map` (or any name you like)
5. Description: "Interactive running race map"
6. Make it **Public**
7. Check "Add a README file"
8. Click "Create repository"

## Step 3: Upload Your Files
1. In your new repository, click "Add file" â†’ "Upload files"
2. Drag and drop these files from your folder:
   - `index.html`
   - `menu-button.js`
   - `app.js` (if it exists as a separate file)
   - `styles.css` (if it exists as a separate file)
   - `races.js` (if it exists as a separate file)
   
   **DO NOT upload:**
   - `index.html.backup`
   - `index_v1.html`
   - `index_clean.html`
   - `START_SERVER.bat`
   - `INSTRUCTIONS.txt`

3. Scroll down and click "Commit changes"

## Step 4: Enable GitHub Pages
1. In your repository, click "Settings" (top menu)
2. Scroll down and click "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Under "Branch", select "main" (or "master")
5. Click "Save"

## Step 5: Get Your Live URL
1. Wait 1-2 minutes for deployment
2. Refresh the Pages settings page
3. You'll see: "Your site is live at https://[your-username].github.io/my-running-map/"
4. Click that URL to view your live running map!
